In the magical pre-COVID days Rob and I began writing a short book that would form an introduction to phylogenetic comparative methods (PCMs). This project was abandoned. However, I find these materials are useful to new users of PCMs, so it seemed like a good time to finally get them online with a DOI so people can use and cite them.

Accompanying online practical exercises in R are available [here](https://nhcooper123.github.io/pcm-primer-online). Please cite these as:

> Natalie Cooper. 2026. Introduction to Phylogenetic Comparative Methods in R. https://nhcooper123.github.io/pcm-primer-online. DOI: 10.5281/zenodo.19134985.
